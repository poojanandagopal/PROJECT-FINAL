Software - Installation Guide 

Step 1:
 Download the Software exe from the website or Insert the CD into CD Rom

Step 2:

Extract the downloaded Zip format file and run the setup.exe file to install the software.

Step 3:
Follow the onscreen instruction by clicking the "Next" button and "Install" Button. 

Step 4: 
Default file location will be in C:\Lynxvision folder in your hard disk.

Step 5:
Once installation is over click on the icon "Lynxvision.exe" from the Desktop.

Step 6:
This software will run for a trial of 30 days. Due to the copy right of the software, this will not run after 30 days of trial unless you get the Software Activation Key from the Manufacture.
